# Semester 2 Project

A small medical delivery company operates daily in an urban and peri‑urban environment. As the company grows, optimizing delivery routes becomes increasingly complex. This prototype app aims to provide a comprehensible dashboard to help the company's CEO make daster decisions grounded on various factors (driver availability, road conditions, types of deliveries...)
